# Google-Play-APK-AAB-Extractor
Google Play APK/AAB Extractor

This is a bash script meant to be run on an Ubuntu terminal. 

### To execute:

#### Ensure you have the following dependencies:

- adb
- bundletool
- java 17
- aapt

#### Ensure you have a phone connected (Tested with Samsung A11)

- USB cable
- Developer mode on
- USB debugging on
